Steps to Run Application:-


1. Install nodejs. Check if it is installed properly using npm --v.

2. Run cmd in Admin Mode. 

3. Browse to path <Dir_Path>\Telefonica_Demo\Backend and run following commands:-
	'npm install' // install all dependencies
	'npm install -g nodemon' // install nodemon package globally
	'nodemon' // run the server on port 6080

4. Run index.html file in <Dir_Path>\Telefonica_Demo\FrontEnd to get to the UI.
